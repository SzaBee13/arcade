# Arcade Gate

A mini-game lobby and persistent player profiles.

## Games

- **Barricade** — Race to the opposite side while sealing routes with tactical barricades. Play vs bot AI or invite friends.
- Nova Drift *(coming soon)*
- Rune Match *(coming soon)*

## Tech

- **Framework** — Next.js 16 with App Router
- **Auth** — Szabee ID (OAuth 2.0 + PKCE)
- **Styling** — Tailwind CSS v4
- **Runtime** — Node.js (custom server via `tsx`)
